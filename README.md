# ⚡ 简介
基于 JavaScript 沉淀的Utils函数库

# 🌱 特性
- JavaScript: 前端三件套之一
- router: 路由
- store: 状态管理方案

# 🍰 功能
- router
- store

# 🚀 开发
```shell
# 配置
1. 一键安装 .vscode 目录中推荐的插件
2. node 版本 16+
3. pnpm 版本 8.x

# 克隆项目
git clone https://github.com/chuxin-cs/utils

# 进入项目目录
cd utils

# 安装依赖
pnpm i

# 启动服务
pnpm dev
```

# 📦️ 多环境打包
```shell
# 构建正式环境
pnpm build
```
